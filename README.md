# Nutrition Club - Attendance Management System

A web application for tracking member attendance with Google Sheets as the database.

## 🚀 Quick Start

**Open `SETUP_GUIDE.md` and follow the steps.**

You've already completed the first part. Now you need to:
1. Deploy the Apps Script (3 min)
2. Update config.js (30 sec)
3. Host your website (2-5 min) - Choose from Netlify, GitHub Pages, or Vercel

**Total time: ~10 minutes**

**Want to host as a public website?** See `HOSTING_OPTIONS.md` for comparison of free hosting platforms.

## 📋 Features

- Login system (default: admin/admin123)
- Dashboard with 4 category boxes
- UMS attendance tracking
- All data loaded from Google Sheets
- Real-time search and statistics
- Responsive design for mobile/desktop

## 📁 Key Files

- **SETUP_GUIDE.md** - Step-by-step setup instructions (READ THIS!)
- **index.html** - Launch this file to start the app
- **config.js** - Update with your Web App URL
- **Code.gs** - Deploy this to Google Apps Script
